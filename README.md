# Simai Convert
Converts a simai chart to import into a Scratch maimai simulator: https://scratch.mit.edu/projects/1270407926/
You can use the web version here: https://eth4nplays.github.io/simai_convert/ or follow the instructions below to use the Python version.

## Installation Guide
- Clone this repo
- Install dependencies: `pip install re`, `pip install csv`

## Usage guide
- Create a new file named `chart.simai` in the same folder and paste the simai score data into it
- Run the Python program
- There will be 4 different data printed
- Open the Scratch project, select `add chart` and type in the chart name
- Paste the data printed accordingly based on the labels
